Flutter 集成百度语言SDK流程

准备工作

  • 百度AI开放平台注册账号，并创建包含语音识别的应用，后面集成要用到AppID，API Key， Secret Key 。
    地址： https://ai.baidu.com/?track=cp:aipinzhuan|pf:pc|pp:AIpingtai|pu:title|ci:|kw:10005792
  • 下载百度语音SDK并解压待用。 地址： https://ai.baidu.com/sdk#asr
  • 创建flutter项目

创建Android端的module

1.打开Flutter项目中的Android目录，打开MainActivty.kt, 点击编辑器右上角Open for Editing in Android Studio选项，
首次打开新窗口会build一段时间，等待即可

2.在新窗口中一次选择File—> New —> New Module –> Android Library，
弹出的窗口中填写 Library信息，一般Module name 填写asr_plugin即可

创建成功后左侧导航栏显示如下：

在asr_plugin里面配置SDK

找到解压好的SDK中的core文件夹
1.将core\libs\bdasr_V3_20210521_0ddfe93.jar复制到flutter项目中的Android\asr_plugin\libs下
2.将core\src\main\jniLibs文件夹整体移动到Android\asr_plugin\src\main下

注意：

  • 导入jar包后单击右键选择Add as Library加载它
  • 如果只需要在线语音识别，那么可以将jniLibs下五种架构中只保留libBaiduSpeechSDK.so和libvad.dnn.so
